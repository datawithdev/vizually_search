How this works?

1. What is Vizually Search?

Vizually search aims to provide a niche search engine for data visualization & business intelligence. The search engine code is open sourced and powered by Google Custom Search. We strive to maintain vendor neutrality and do not include direct links to commerial vendors. If you would like to be included in the search, please email me at dev @ vizually.io

2. How was it developed?

The source code uses Twitter Bootstrap, Javascript, and CSS. There is no server-side code.

3. Why beta?

Vizually search uses white-labeled search API from Google. This API is not free to use. Large number of API calls may hit the daily limit.
